About Typing Rain

Mission

Typing Rain turns any weekly spelling list into a falling-word typing game.

It is built for parents, teachers, homeschool families, and ESL tutors who need quick spelling practice with missed-word replay and no student accounts.

About the Developer

Hi! I'm an indie developer who enjoys building small, polished web experiences. I care about usability, accessibility, and language friendliness.

Tech Stack & Features

  • Fast vanilla JavaScript with modular architecture
  • Optimized canvas rendering and particle effects
  • Custom spelling list input
  • Missed-word replay and shareable practice links
